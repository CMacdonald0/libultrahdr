## Getting Started

Supports:
- x86_32/x86_64 on Linux.

Preferred Compiler: Clang
Tested with:
- C compiler: Clang 15.0.7
- CXX compiler: Clang 15.0.7

## Building Commands

### For x86_32 on a x86_64 Linux machine
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```
### For Mac OS
```
$ mkdir build
$ cd build
$ cmake ..
$ make
```
